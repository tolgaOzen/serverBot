## usage

#### path, url and  program enter config.toml file

```toml

[url]
URL="http://127.0.0.1:3735"

```

> ip address of server

```toml

[program]
PROGRAM="./exampleProgram"

```

> program to run


```toml

[path]
PATH="/Users/tolgaozen/_GOPROJECT/src/example"

```

> path of the program to run


#### and run program
